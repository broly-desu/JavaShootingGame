期末課題「シューティングゲーム」

------------------------------------------------------------------------------------------------------------------------

ルール

	このゲームを起動するとすぐにゲームが開始され、画面上にいる敵をすべて倒すことでステージクリアとなります。
	
	ステージが進むごとに敵の使う弾の性能があがり、速度が上昇していきます。
	
	自機は一度でも被弾すると破壊され、ゲームオーバーになりますが、敵よりも当たり判定が小さくなっています。
	
	敵の種類については後ほど説明します。

------------------------------------------------------------------------------------------------------------------------

操作方法

　※ゲーム内
	
	←　→キー：左右移動
	
	Spaceキー : 弾発射

　※ゲームオーバー時

	Enterキー : 最初から

　※ゲームクリア時
	
	Enterキー : 次のステージへ進む

------------------------------------------------------------------------------------------------------------------------

敵の種類

	敵は六種類います。
	
	1.EnemyBase : 敵の基地です、敵を生み出す能力を持っています。
	
	2.StraightEnemy : 一定の速度で下に落ちてきます。
	
	3.DropEnemy : T字型の敵です、StraightEnemyと同じく落下してきますが、落下速度がだんだんと上昇します。
	
	4.RandomEnemy : 一定の速度で落ちてきますが、左右にランダムで動きます。
	
	5.CurveEnemy : 一定の速度で落ちてきますが、プレイヤーのいる方向へ誘導してきます。
	
	6.BossEnemyBase : 5ステージに一回だけ現れる大型基地です、ステージ開始時にEnemyBaseを下方向に二体召喚し、自らも敵を
	
	召喚してきます。EnemyBaseより大型ですが、体力、速度、そして敵の召喚頻度すべてにおいてその赤さにふさわしい3倍です。
	
	まれに赤いビームを３方向に４発発射してきます、ビームを撃墜することは可能ですがあまりに高速なため注意してください。

------------------------------------------------------------------------------------------------------------------------

敵の体力一覧
	
	EnemyBase : 10
	
	StraitEnemy : 5
	
	DropEnemy : 2
	
	RandomEnemy : 2
	
	CurveEnemy : 3
	
	BossEnemyBase : 30
	
------------------------------------------------------------------------------------------------------------------------

ステージ進行による強化について

	敵の速度や体力は、ステージを進むごとに増えていきます。
	
	速度はステージが進むごとに1ずつ増えていきますが、体力はステージが10個進むごとにその体力分増えていきます。
